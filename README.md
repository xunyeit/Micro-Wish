## 一、产品概述
**“重庆微心愿”**** ****是一款由重庆市相关单位或社会组织发起的城市级****公益+社交融合型移动应用****，旨在通过“发布心愿—认领心愿”的轻量化互动机制，连接市民之间的善意与资源，推动邻里互助、情感联结和社区共治。**

**它以“****点亮微心愿，通往幸福站****”为品牌口号，聚焦于满足市民在生活、情感、兴趣等方面的****小而真、暖而实的愿望****，如获取一件心仪物品、一次旅行体验、一个宠物陪伴机会等。用户可通过微信登录快速参与，实现从“表达需求”到“资源对接”的闭环服务。**

**该平台不仅是一个愿望传递工具，更是一个城市温情网络的构建者，将个体孤独感转化为社会支持力，打造有温度的城市数字生态。**

## 二、核心功能
### 1. **首页概览：信息聚合与视觉引导**
+ **首页顶部展示主题横幅：“点亮微心愿，通往幸福站”，配合卡通熊猫IP形象（象征重庆文旅元素），营造温暖亲切的品牌氛围。**
+ **中部显示关键数据指标：**
    - **未点亮：239个**
    - **点亮中：58个**
    - **已点亮：22个**
    - **总心愿数：2100+**
+ **下方设置六大核心功能图标入口：**
    - **生活交友**
    - **美食探店**
    - **浪漫旅行**
    - **宠物之家**
    - **旧物置换**
    - **职业发展**

### 3. **交互流程示例：发布 & 认领心愿**
#### （1）发布心愿流程：
1. **用户点击“发布心愿”**
2. **填写：**
    - **姓名（必填）**
    - **心愿标题（必填）**
    - **心愿类型（选择类别）**
    - **联系方式（手机号/微信）**
    - **心愿说明（可选，用于描述动机或细节）**
3. **提交后进入“心愿池”，等待被认领**

#### （2）认领心愿流程：
1. **用户浏览“生活交友”或其他板块的心愿列表**
2. **点击某个心愿查看详情**
3. **查看对方联系方式（如：187****7989）**
4. **点击“认领心愿”按钮**
5. **系统自动通知原发布者，双方可通过私信或电话联系达成对接**

### 4. **身份认证与安全机制**
+ **登录方式：仅支持****微信授权登录****，确保身份真实可信**
+ **信息保护：联系电话默认只显示前三位+后四位，中间星号隐藏**
+ **内容审核：平台对发布内容进行关键词过滤与人工抽检，防止诈骗或不当言论**
+ **反馈通道：用户可在“我的”页面提交问题或建议**

## 三、产品价值
### **用户价值：连接需求与善意，构建城市温情网络**
+ **满足个性化小愿望****：无论是想拥有一个宠物、一次旅行、还是一次深度交流，都能找到对应渠道。**
+ **降低社交门槛****：通过“微心愿”形式打破陌生人之间的隔阂，让交友变得更自然、更有目标感。**
+ **提升幸福感与归属感****：当别人帮你实现愿望时，你会感受到城市的温度；当你帮助别人时，你也收获了成就感。**
+ **增强城市认同感：以“重庆”命名，结合地方文化符号（如轨道交通、熊猫IP），强化本地居民的情感连接。**

## 架构特点
### 架构特性
+ **前后端分离架构，独立开发，符合主流开发模式**
+ **前端以Vue3+Vite为主技术，AntdV为UI界面框架**
+ **后端SpringBoot3为基础，MybatisPlus为数据操作框架，Redis为缓存框架**
+ **Maven多模块管理，插件化开发，方便安装、卸载、升级，降低耦合**
+ **业务模块与API抽离，模块之间便捷引用**
+ **数据库设计精巧，字段规范、易于扩展**
+ **支持国产密码算法加解密，等保测评国产项目无压力**
+ **支持MYSQL、ORACLE、SQLSERVER、PGSQL等主流标准结构式数据库**
+ **支持达梦、人大金仓、南大通用、九有、瀚高、虚谷数据库等国产数据库**
+ **支持中创、宝蓝德、东方通等中间件**
+ **支持Windows、Linux操作系统、国产操作系统部署**

### 功能特性
+ **完善的系统基础功能，满足使用需求，避免重复造轮子**
+ **支持本地文件、阿里云文件、腾讯云文件、MINIO文件上传**
+ **支持本地邮件、阿里云邮件、腾讯云邮件发送**
+ **支持阿里云短信、腾讯云短信发送**
+ **B、C端双账号认证体系，会话治理各自独立**
+ **完善的登录日志、操作日志、异常日志等审计功能**
+ **完善的会话监控、数据源监控、系统监控等必备监控功能**
+ **支持组织机构、权限管理、定时任务、系统配置等基础功能**

### 安全特性
+ **采用SaToken轻量级 Java 权限认证框架，功能强大、学习成本低**
+ **支持登录认证、权限认证、单点登录、三方登录、OAuth2.0等认证模式**
+ **增强的RBAC权限设计，资源于接口独立授权，更加灵活**
+ **支持按钮级别细粒度独立授权，界面按钮动态展示**
+ **支持API接口注解式、路由拦截式鉴权，防止越界访问**
+ **独创的数据范围机制，每个接口都可以配置不同数据范围**
+ **支持限流防抖，防重复提交，有效阻止脏数据产生**
+ **密码、手机、身份证号等使用国密算法加密传输、加密存储，数据更安全**
+ **操作日志使用SM2进行完整性保护，满足安全审计要求**

### 界面特性
+ **Vue3 + Vite为基础，AntdV为界面UI框架，视觉风格清新简洁**
+ **精细化设计，注重界面的每一处细节，操作轻松友好**
+ **暗黑风格、经典菜单、双排菜单、多页签、目录坞、主题切换等功能应有尽有**
+ **统一的网络框架、API接口拦截框架，拿来即可上手**

## 运维环境
+ **全系列系统服务器。推荐使用 Centos 7.6**
+ **2 核 2 G 100MB 3M带宽 为最低配置，建议 2 核 4G 100Mb 5M带宽。**
+ **Node.js 14x**
+ **Java1.8**
+ **Mysql 5.7**
+ **Redis 6.2.8**
+ **阿里云 OSS – 可更换其它。**
